# Archived GUI scripts

This folder preserves earlier LIDDIA GUI experiments and flat-file versions for reference.

The active GUI is now the modular app in `../liddia_gui_app/liddia_gui/` and should be launched with:

```console
cd ../liddia_gui_app
python -m liddia_gui.app
```

Archived files here are not the recommended entrypoint for new work.

## Contents

- `liddia_gradio_app*.py`: earlier Gradio GUI versions and flat-file references.
- `liddia_nicegui_app.py`: earlier NiceGUI experiment.
- `debug/`: scratch scripts preserved from cleanup.
